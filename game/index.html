1GAM 2013

OneGameAMonth is a year long game jam meant to help you level up your gamedev skills. The overall goal is to build one game every month for a year. I participated in 2013 and gave myself the additional goal of trying out a different gamedev tool every month. In the years before 2013, I had spent my free time perusing the internet looking for the latest and greatest gamedev tools. Unfortunately, I reached a point where I was spending more time collecting gamedev software than actually using it. I wanted to change this, but couldn't find the motivation to do it. Then I heard about this game jam challenge and off I went. I even managed to make it onto the high score leaderboard. I think I might still be there.

You'll notice that there are only 11 games in this list. I attempted to build a game in June using LimeJS. It was a nightmare to compile and get it to work on my own machine, so I have zero confidence that it will work on anyone else's.
